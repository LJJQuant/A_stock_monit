指标公式
全部都是主板策略。
1.{KDJ J值少于98 AND今天的J值大于昨天的J值 }
当天实时的判断【预警】，收盘时候满足【回测】。
2.{连续3天J小于K }
今天，昨天，前天都满足【预警+回测】
3.{今天的换手率除以昨天的换手率大于1.28倍，且小于12倍 }
【预警 + 回测】
4.{当日换手率大于3.28% 且小于25.8%}
【预警 + 回测】
5.{今天的隔日换手率大于昨天的隔日换手率或者今天的隔日换手率大于2.5倍}
隔日换手率公式【今天换手率 /昨天换手率】。满足其一就可以。
6.{流通值大于16亿，且小于500亿 }
流通市值，盘中可能会变化。
7.{成交额大于1.68个亿 小于36.68亿 }
8.{获利筹码大于78% }  【同花顺公式不用加这个】，等待处理，先不加
9.{前6日或者前7天涨幅为11个点以内 }
【不包括今天，使用收盘价计算】
比如今天是1.10日。1.2 ~1.9 的涨幅就是七天涨幅【交易日】
10.{当日涨幅在4-9.5个点 }
11.{当前价除以均价小于4% }
【当天实时有，或者计算】。回测当前价为收盘价，均价是成交额/ 成交数量。
12{当前价为10日内最高价}
【预警是实时价格，回测是收盘价】
13{当前价大于60天均价，均线排列：MA5>MA20 AND MA5 > MA60}
A：当前价 > MA(60)，回测使用收盘价，算均价都必须使用今天价格。
B:当天用实时，回测用收盘价，且都算当天
14{连续5日收盘价涨幅均小于9.5% }
不包括当天，过去五个交易日的的涨幅都小于。
15.{不能出现4连阳}
不包括当天。1.5看的话，1.1 ~ 1.4不能同时出现收盘 > 开盘价
16.{10天内没有大跌超过-8%（收盘价除以开盘价）的交易日}
不包括当天，实体涨幅
17.{检查前2日（不含今日）不能出现大于4%的上影线}
限制必须为上影阳线，阴线不考虑。
18.{筹码集中度大于昨日}【爬虫，到时可能考虑提前一天落库】
19. {当日增仓比例大于6%}【主力净流入净占比】
20. {尾盘30分钟指标满足其一即可}【当天实时，回测收盘】
倍数指的是2：30 以后的成交额，跟2：00到2：30的成交额。满足任一即可。

   D . 0.9 倍换手率大于4000W
